# Cloud-Based Task Management System

## Overview
This project is a cloud-based task management application developed using Java, React.js, Node.js, PostgreSQL, and AWS services.

## Features
- User authentication
- CRUD task operations
- RESTful APIs
- Cloud deployment using AWS EC2
- Secure storage with AWS S3 and IAM

## Technologies Used
- Java
- React.js
- Node.js
- PostgreSQL
- AWS EC2
- AWS S3
- Docker

## Purpose
This project was built to improve task tracking and management through scalable cloud-based architecture.
